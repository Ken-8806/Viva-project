📌 Project Description

The Virtual Classroom Dashboard is a web-based learning platform that allows teachers and students to interact online. It supports class management, assignments, and real-time chat between users in a simple and interactive interface.

🚀 Features
🔐 Login system for Teachers and Students
📚 Create and manage classes (Teacher role)
📝 Add and view assignments
💬 Real-time class chat system
📊 Role-based dashboard (Teacher / Student)
💾 Data stored using localStorage (no backend required)
📱 Responsive and user-friendly UI
🛠️ Technologies Used
React.js (Frontend Framework)
JavaScript (ES6)
HTML5 & CSS3
LocalStorage (Data handling)
📂 Project Structure
src/
 ├── App.jsx
 ├── App.css
 ├── index.js
⚙️ How to Run the Project
Clone or download the project
Open terminal in project folder

Install dependencies:

npm install

Run the project:

npm start

Open browser and go to:

http://localhost:5173
